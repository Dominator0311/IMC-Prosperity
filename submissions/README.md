# Submissions

This directory is intentionally a public placeholder while the live competition
is in progress.

The repository currently publishes the reusable research engine, backtesting
harness, public historical-data analysis, and documentation. Final competition
submission bundles and official simulator/result artifacts are kept out of the
public tree until the competition has concluded.

Planned release after final lock:

- final Round 5 submission bundle;
- concise official-result summary;
- selected archived submissions where they are useful for learning;
- reproducibility notes for replaying public historical data.
