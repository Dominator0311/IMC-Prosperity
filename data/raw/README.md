# Public Raw Data

This directory contains public Prosperity replay CSVs used by the local
research and backtest harness.

- `tutorial_round_1/` contains tutorial-round replay data.
- `round_1/` through `round_5/` contain public historical round replay
  data when available.
- Stress fixtures may appear in separate `*_stress/` directories.

Private simulator results, generated outputs, final submissions, and
live-round strategy variants are intentionally kept out of Git until
they are ready for public release.
